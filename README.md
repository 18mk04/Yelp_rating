# Yelp Rating Prediction + Two-Dashboard App

This repo contains:

- `notebooks/` — Task 1: prompt evaluation notebook (rating_prompt_eval.py)
- `backend/` — FastAPI backend, DB and Gemini wrapper
- `user_dashboard/` — Streamlit user-facing app
- `admin_dashboard/` — Streamlit admin app
- `data/` — place `yelp.csv` (download from Kaggle)
- `docs/` — supporting docs and report.pdf

See `.env.example` for environment variables. Follow the instructions in notebooks/README or this README to run locally.
